
# [Portfolio website](https://github.com/ziuteczek/portfolio)

A modern, fast, and SEO-friendly portfolio website built to showcase my projects, experience, and interests.  
The site is generated using a **Static Site Generator (SSG)**, providing excellent performance and discoverability.

---

## Tech Stack

- **[Astro](https://astro.build/)** – Static Site Generator for lightning-fast performance  
- **[Tailwind CSS](https://tailwindcss.com/)** – Utility-first CSS framework for sleek and responsive design  
- **[Page CMS](https://pagecms.com/)** – Simple and flexible content management  

---

## Features

- **Blazing fast** page loads  
- **SEO-optimized** with metadata and structured content  
- **Responsive design** that works on any device
